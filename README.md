## Installation

SparkNetManager is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'SparkNetManager', :git => "https://github.com/SparkeXHApp/SparkNetManager.git"
```
